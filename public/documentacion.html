Documentación Katana

Framework PHP para desarrollo ágil

1. Configuración de entorno y acceso local

Katana se ejecuta en http://katana.local mediante configuración de Virtual Host.

<VirtualHost *:80>
    DocumentRoot "C:/xampp/htdocs/katana/public"
    ServerName katana.local
</VirtualHost>

Además, editar el archivo C:\Windows\System32\drivers\etc\hosts:

127.0.0.1    katana.local

3. Definir Rutas

// routes/web.php
$router->get('/', [HomeController::class, 'index']);
$router->post('/login', [AuthController::class, 'login']);

5. Vistas con Layouts

Vista: Resources/Views/home.php

    <h1>Bienvenido a Katana</h1>
<?php endSection() ?>
<?php renderLayout('main') ?>

6. Conexión a Base de Datos

Configura el archivo .env:

DB_HOST=127.0.0.1
DB_NAME=katana_db
DB_USER=root
DB_PASS=

Obtén instancia:

$db = \Src\Core\Database::getInstance();
